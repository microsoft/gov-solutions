Getting Started

  1. Explore Personas and Use Cases

    Get familiar with common government scenarios and the roles involved. These give context on where the app templates and data models can help.

  2. Review Data Models and App Starter Kits

    Browse the standardized data models and prebuilt starter apps. Each includes overviews, release notes, and documentation to guide setup.

  3. Download and Deploy

    Select the latest release of the data model or app kit you need, and follow the how-to documentation to deploy in your environment.
